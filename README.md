# DSA Mastery Tracker

A high-performance, dark-themed DSA problem-solving tracker with hand-picked LeetCode problems, categorized by topic and difficulty.

## Features
- **129 Hand-picked Problems**: Curated across key topics like Arrays, Strings, DP, Graphs, and more.
- **Dynamic Progress Tracking**: Real-time solved counters and progress bars.
- **Intelligent Streak Logic**: Automatically tracks consecutive days of activity based on problem timestamps.
- **Filter & Search**: Easily find problems by difficulty (Easy, Medium, Hard) or name/pattern.
- **Local Storage**: Your progress is saved automatically in your browser's local storage.

## How to Run
Simply open `dsa-leetcode-tracker.html` in any modern web browser or use a local server (like VS Code Live Server).
