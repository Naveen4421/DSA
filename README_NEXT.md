# DSA Mastery Tracker (Next.js Version)

This is the modernized version of the DSA Tracker built with Next.js, Tailwind CSS, and Framer Motion.

## Features
- **Next.js & React**: Modern framework for high performance and better developer experience.
- **Tailwind CSS**: Utility-first styling for a premium look.
- **Framer Motion**: Smooth, buttery animations and transitions.
- **Supabase Auth & DB**: Real-time cloud sync for your progress.
- **Mobile Responsive**: Fully optimized for all screen sizes.

## How to Run Local

1.  Make sure you are in the `dsa-tracker` directory.
2.  Install dependencies (if you haven't):
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment on Vercel
1.  Push this code to a GitHub repository.
2.  Import the repository into Vercel.
3.  Add the Environment Variables from `.env.local` to your Vercel project settings.
4.  Deploy!
